Chromatics - Pygame Color Module

Chromatics module is a color picker to use with Pygame. The API documentation to use the module is currently in the docstrings. The module is in early development.

The chromatics-app.py can be used as a demo of the module. To use, select color hue from color spectrum, then select color from colormap. The selected color is displayed along with the RGB value, click the selected color to copy RGB value to the clipboard.
